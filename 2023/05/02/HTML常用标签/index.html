HTML 链接

在 html 中用 a 标签来定义一个链接，链接简单理解就是一个网址，通过点击该标签框住的文字可以跳转到相应的网页网站等，这里通过 a 标签内的一个 herf 属性来表示即将要跳转的链接地址, 语法如下： 当然如果暂时不想跳转可以定义 herf 属性值为“#”，如: 锚点链接：目标位置添加一个 id href="#(id 名字)"，作用跳转到该 id 元素所在的位置。标签 a 链接中的 target 属性表示可以使用该属性定义链接的文档 (网站网页等) 在何处显示，就是通过哪种方式打开的意思。target 的属性值有 _self,_blank,_parent,_top。_self 表示默认跳转方式，在原来的窗口打开(在同一框架或窗口中打开所链接的文档)，_blank 表示 在新浏览器窗口中打开链接文件，_parent 将链接的文件载入含有该链接框架的父框架集或父窗口中。_top 在当前的整个浏览器窗口中打开所链接的文档，因而会删除所有框架。

HTML 表格

在 html 中用 table 标签来定义一个表格，table 标签中有若干行用 tr 标签来表示，每一行分成若干份表示每个单元格，用 td 标签来表示。表格还可以更规范一些，用 thead 标签表示表格的表头，表头中标题用 th 标签表示，tboody 标签表示表格的内容主体，tfoot 表示表格的页眉。实例如下： td 中的属性 rowspan=“合并单元格行数” 跨行合并 colspan=“合并单元格列数” 跨列合并。合并之后还需要减去 (删除) 合并的单元格。

HTML 区块

HTML 中用 div 与 span 来把元素组合起来，大多数 HTML 元素被定义为块级元素或内联元素。
块级元素在浏览器显示时，通常会以新行来开始（和结束）实例：h1,p,ul,table
内联元素在浏览器显示时，通常不会以新行开始，实例：b,td,a,img
HTML 中 div 标签是块级元素，它可以用于组合其他 HTML 元素，是一个容器，方便用于文档布局，设置样式等，默认独占一行。
span 标签是内联元素，它可以用作为文本的容器，没有特殊含义，与 css 一起使用时可以为部分文本设置样式属性，默认为内容多大就站多大。
